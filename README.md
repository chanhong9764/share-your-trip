## 🖥️ 프로젝트 소개
- 친구 혹은 지인들과 서로의 니즈를 실시간으로 공유하며 여행 일정을 계획하고 수정할 수 있도록 도와주는 플랫폼입니다.
- 방문했던 여행지 중에 인상 깊었던 곳을 다른 유저들과 공유하고 소통하여 여행 정보를 얻을 수 있습니다.

- `Java 8`
- `JDK 1.8.0`
- **IDE** : STS 3.9
- **Framework** : Springboot(2.x)
- **Database** : Oracle DB(11xe)
- **ORM** : Mybatis
